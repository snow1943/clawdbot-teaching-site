## 🎯 关于本项目

本项目是 Clawdbot (OpenClaw) 的官方教学网站，提供从入门到精通的完整教程，包括：

- ✨ 详细介绍 Clawdbot 的核心特性
- 🚀 多平台部署指南（本地、Docker、云服务器、云原生）
- 📚 详细的实操教程和配置说明
- 🏗️ 技术架构解析
- ❓ 常见问题解答

## 🚀 快速访问

网站已部署在 GitHub Pages：[https://clawdbot.github.io/teaching-site](https://clawdbot.github.io/teaching-site)

## 🛠️ 本地开发

```bash
# 克隆项目
git clone https://github.com/clawdbot/teaching-site.git
cd teaching-site

# 使用任意静态服务器运行
# 方法1: Python
python -m http.server 8080

# 方法2: Node.js (http-server)
npx http-server -p 8080

# 方法3: VS Code Live Server 插件
```
