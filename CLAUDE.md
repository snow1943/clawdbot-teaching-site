# CLAUDE.md

This is the official teaching site for Clawdbot/OpenClaw.

## Project Overview

A static website teaching users about Clawdbot - an open-source AI assistant framework.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom variables, animations, responsive design
- **JavaScript** - Vanilla JS for interactivity

## Development Commands

```bash
# Serve locally
python -m http.server 8080
# or
npx http-server -p 8080

# Build (if using a build tool)
npm run build
```

## Key Files

- `index.html` - Main page with all sections
- `styles.css` - Complete styling
- `script.js` - All interactive features
- `README.md` - Project documentation

## Features Implemented

- Responsive navigation with smooth scroll
- Hero section with animated gradients
- Feature cards grid
- Tabbed installation guide (Local/Docker/Cloud/Cloud-Native)
- Guide cards
- Architecture diagram
- FAQ accordion
- Code copy functionality
- Intersection Observer animations
- Parallax mouse effects

## Deployment

The site is designed to be deployed as a static site:
- GitHub Pages
- Vercel
- Netlify
- Any static hosting
