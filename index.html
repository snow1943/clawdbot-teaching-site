安装部署

选择你的部署方式

无论你使用什么环境，总有一种方式适合你

01

环境准备

确保你的系统已安装 Node.js 18+ 和 Git

# 检查 Node.js 版本
node -v  # 需要 v18.0.0 以上

# 检查 npm 版本
npm -v

# 如果没有安装 Node.js，推荐使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
02

克隆项目

从 GitHub 克隆 Clawdbot 源码

# 克隆项目
git clone https://github.com/clawdbot/clawdbot.git
cd clawdbot

# 切换到最新稳定版本
git checkout main
03

安装依赖

使用 npm 安装项目依赖

# 安装依赖（使用 pnpm 更快）
npm install -g pnpm  # 如果没有 pnpm
pnpm install

# 或者使用 npm
npm install
04

启动服务

启动 Clawdbot 服务

# 开发模式启动（支持热重载）
npm run dev

# 生产模式启动
npm run build
npm start

# 🎉 服务已启动，访问 http://localhost:3000
01

环境准备

确保已安装 Docker 和 Docker Compose

# 检查 Docker 版本
docker --version  # 需要 20.10+

# 检查 Docker Compose
docker compose version

# 启动 Docker 服务（Linux）
sudo systemctl start docker
sudo systemctl enable docker
02

创建配置文件

创建 docker-compose.yml 配置文件

version: '3.8'

services:
  clawdbot:
    image: clawdbot/clawdbot:latest
    container_name: clawdbot
    restart: unless-stopped
    ports:
      - "3000:3000"
    volumes:
      - ./data:/app/data
      - ./config:/app/config
    environment:
      - NODE_ENV=production
      - TZ=Asia/Shanghai
    networks:
      - clawdbot-network

networks:
  clawdbot-network:
    driver: bridge
03

启动容器

使用 Docker Compose 启动服务

# 创建数据目录
mkdir -p data config

# 启动服务
docker compose up -d

# 查看日志
docker compose logs -f

# 🎉 访问 http://localhost:3000
04

更新部署

更新到最新版本

# 更新镜像
docker compose pull

# 重启服务
docker compose up -d

# 清理旧镜像
docker image prune

腾讯云

推荐配置：2核 4GB

# SSH 连接到服务器
ssh root@YOUR_SERVER_IP

# 安装脚本一键部署
curl -sSL https://get.clawd.bot/tencent | bash

# 或手动安装
wget -O install.sh https://get.clawd.bot/install.sh
chmod +x install.sh
./install.sh

阿里云

推荐配置：2核 4GB

# SSH 连接到服务器
ssh root@YOUR_SERVER_IP

# 安装脚本一键部署
curl -sSL https://get.clawd.bot/aliyun | bash

# 或使用 Docker 部署
docker run -d -p 3000:3000 \
  -v /opt/clawdbot/data:/app/data \
  clawdbot/clawdbot:latest

火山引擎

推荐配置：2核 4GB

# SSH 连接到服务器
ssh root@YOUR_SERVER_IP

# 使用 Docker Compose
mkdir -p /opt/clawdbot
cd /opt/clawdbot

cat > docker-compose.yml << 'EOF'
version: '3.8'
services:
  clawdbot:
    image: clawdbot/clawdbot:latest
    restart: unless-stopped
    ports:
      - "3000:3000"
    volumes:
      - ./data:/app/data
EOF

docker compose up -d
01

Kubernetes 部署

使用 Helm Chart 快速部署到 K8s 集群

# 添加 Helm 仓库
helm repo add clawdbot https://charts.clawd.bot
helm repo update

# 安装 Clawdbot
helm install clawdbot clawdbot/clawdbot \
  --namespace clawdbot \
  --create-namespace \
  --set replicaCount=2 \
  --set service.type=LoadBalancer
02

云函数部署

部署到 AWS Lambda / 阿里云函数计算

# 使用 Serverless Framework
serverless deploy \
  --provider aws \
  --template clawdbot-template

# 阿里云函数计算
funcraft deploy
03

Docker Swarm 部署

使用 Docker Swarm 进行集群部署

# 初始化 Swarm 集群
docker swarm init --advertise-addr YOUR_IP

# 部署服务
docker stack deploy -c stack.yml clawdbot

Clawdbot 是开源的、本地优先的 AI 助手框架。与闭源产品不同，你的数据完全由自己掌控。它支持 50+ 平台接入，拥有强大的插件系统，且完全免费。同时，社区活跃，持续更新，确保你始终获得最好的体验。

Clawdbot 完全免费开源，采用 MIT 许可证。你只需要支付服务器成本（如果选择云端部署）。AI 模型的 API 调用费用由你使用的模型提供商收取（如 OpenAI、Anthropic 等）。

Clawdbot 采用本地优先架构，所有数据默认存储在本地。你可以完全控制数据的存储位置和传输方式。我们不会收集任何用户数据，也不会向第三方发送你的对话内容。

Clawdbot 支持所有主流大语言模型，包括：Claude（Anthropic）、GPT-4/3.5（OpenAI）、DeepSeek、通义千问、文心一言、GLM、Kimi 等。你可以同时配置多个模型，按需切换。
